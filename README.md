# Soccervideo
> "Scorebat Football"의 open API를 통해서 만들었습니다.  
  매일매일 경기에 대한 영상에 관련된 자료가 update됩니다.   
  이를 활용해서 ***최신 축구경기 관련영상을 보여주는 서비스***입니다.

> https://soccervideo.herokuapp.com/ 로 heroku를 통해서 웹호스팅을 진행했습니다.
